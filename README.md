# EnergyOfTales
Energy Of Tales ist ein KI Rollenspiel, gemacht um für die Frage „Was macht KI mit der Gesellschaft?“ zu sensibilisieren.

Für das Spiel benötigt man:
- 3 Würfel
- 3 Spielkarten pro spieler:in
- Diesen Prompt: https://www.prompt-cards.eu/view/cff9d591-4c7e-4f43-9349-cd8c73a8f820
- Eine Chat-KI
- Spielfigur (optional)
- Spielfeld (optional)

Den Rest mach die KI. Die Würfel, die Spielfigur, das Spielfeld und die Box zum Aufbewahren kann ausgedruckt werden. Die .STL Dateien findet man hier.
